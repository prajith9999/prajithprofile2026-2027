# Prajith NS — Professional Portfolio

A modern, professional React portfolio website for **Prajith NS**, Software Engineer based in Doha, Qatar.

## Sections

- **Home** — Hero, expertise overview, and technical skills
- **About** — Professional summary, personal details, education, and career objective
- **Experience** — Work history at CABTECH, VHYPOTENUSE, CloudXgens, and Fortune Systems
- **Certifications** — Credentials, projects, and language skills
- **Contact** — Direct contact form and details

## Tech Stack

- React 19 + Vite
- React Router DOM
- Poppins font (Semi-Bold 600)
- CSS custom properties with glassmorphism

## Run Locally

```bash
cd it-training-app
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
